# **App Name**: StreamVerse

## Core Features:

- Movie Gallery: Display movies in horizontal scrolling rows with a hero banner.
- Remote Video Upload: Upload videos from a remote URL directly to Supabase storage.
- Movie Database: Store movie metadata (title, description, genre, videoURL, thumbnailURL) in the Supabase database.
- User Profiles: Manage user profiles and movie viewing history.
- Crypto Payment Integration: Connect a Web3 wallet to unlock premium movies. Placeholder function for demonstration purposes.
- Adaptive Video Player: Stream videos using React-Player, adapting to different video formats and resolutions.

## Style Guidelines:

- Primary color: Dark indigo (#4F46E5) to establish a modern, serious mood, appropriate to the tech-focused space of streaming entertainment.
- Background color: Near-black (#121212) to reinforce the dark theme.
- Accent color: Violet (#8B5CF6) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif providing a modern and neutral look.
- Use a grid-based layout for the movie gallery and a full-width hero banner.
- Use simple, outline-style icons for navigation and video controls.
- Subtle transitions and hover effects for interactive elements.
